Comparing webmaps in 3x and 4x JSAPI

https://jscore.esri.com/testing/compare-3x-4x-webmaps/daily.html?webmap=901ab4abfc884c408fc6057f2dc53fbc - Plug in your own (production) webmap ID!

  • To test with devext (or other portal), add portal param, e.g., https://jscore.esri.com/testing/compare-3x-4x-webmaps/daily.html?webmap=c03c8685721b4b7a99f0968765277530&portal=https://devext.arcgis.com/
  • To not sync the maps, add sync=false, e.g., https://jscore.esri.com/testing/compare-3x-4x-webmaps/daily.html?webmap=4f0e3f5bed3744788ebc028965a8a24a&portal=qaext&sync=false.

Known differences

    Better with 4x
  • Better performance
  • multi-line labels
  • line labels that curves
  • date labels formatted as dates
  • Labels and Proportional Symbols don't align for multi-part polygons. 3x wontfix: #5080
  • non-black hatch patterns (same as Pro)
  • better attribution: less unnecessary duplication, attribution for operational layers
  • for WMTS, 4x will automatically try KVP if RESTful fails. 3x does not. You have to set the serviceMode - which webmap does automatically, but not the layer
    Needed in 4x for full webmap support (aka 3x compatibiliy)
  • support for temporal renderer - API issue #7351.
  • support for trackRenderer - API issue #9958.
  • support for WFS - Planning issue #12.
    Still better in 3x - maybe needed for almost full webmap support
  • polygon label placement for weird shapes - API issue #17476.
  • better ordering of overlapping symbols of different sizes
  • showing non-CORS symbols without imagedata
  • some other CORS issue
    3x - no plans to fix
  • popup can be maximized to fully view tables within a popup
  • fake italic labels for fonts without italic support, e.g. "Arial Unicode MS" - API issue #20155 (Pro supports italic if installed on client machine, not otherwise).
  • loading old (pre-2.0) webmap versions
    3x - fixed in which 4x version?
  • Fixed in 4.14: support for clustering clustering.
  • Fixed in 4.12: support for timeslider
  • Fixed in 4.15: support for animated markers - #17999.
  • Fixed in 4.16: arrow support on lines.
    Just different...
  • halo on label is skinnier in 4x
  • heatmap looks a little bit different but 4x has better legend :)
